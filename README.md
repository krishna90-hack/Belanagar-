## 🚀 GitHub पर Upload करें

1. GitHub पर नया repository बनाएं: `belanagar-website`
2. सभी files upload करें
3. Commit करें: `"Initial commit: Belanagar official website"`

## ⚡ Netlify पर Deploy करें

### Option 1: Netlify Drop (सबसे आसान)
1. [netlify.com/drop](https://app.netlify.com/drop) खोलें
2. `belanagar-website` folder को drag & drop करें
3. Live link मिल जाएगा!

### Option 2: GitHub से Connect
1. [netlify.com](https://netlify.com) → New Site from Git
2. GitHub → `belanagar-website` repository select करें
3. Build settings: Publish directory = `.`
4. Deploy!

## 🏘️ About

Belanagar (बेलानगर) is a vibrant village in Ward No. 03, Nirmali Panchayat, Pipra Block, Supaul District, Bihar — 852134. Located on the banks of Belanagar River in the Kosi region, it is a center of community unity, cultural heritage, and spiritual awakening.

**Key Landmarks:**
- Belanagar River & Bridge
- Rajdhani Chhath Ghat
- Hariballabh Mandal (Bhagat Baba) Center

**Active Organizations:**
- BNFC — Belanagar Fair Committee
- BN-SPC — Saraswati Puja Committee
- BN-YS — BN Yuwa Sangathan
- BNC — Belanagar Cricket Club (🏆 Raj 11, 2026 Champions)
